# Contributing to De:gram

De:gram への貢献をご検討いただきありがとうございます！

## 開発の進め方
1. このプロジェクトをフォークします。
2. 新しい機能やバグ修正のためにブランチを作成します (`git checkout -b feature/amazing-feature`)。
3. 変更をコミットします (`git commit -m 'Add some amazing feature'`)。
4. ブランチにプッシュします (`git push origin feature/amazing-feature`)。
5. プルリクエストを作成します。

## 問題の報告
バグを発見した場合や新機能の提案がある場合は、GitHub の Issues を通じて報告してください。

## 行動規範
建設的でフレンドリーなコミュニケーションを心がけてください。
